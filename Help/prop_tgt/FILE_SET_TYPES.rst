FILE_SET_TYPES
--------------

.. versionadded:: 4.5

Read-only list of the target's file set types.

See Also
^^^^^^^^

Related properties:

* :prop_tgt:`FILE_SETS_<TYPE>` to list ``PRIVATE`` and ``PUBLIC`` file sets of
  type ``<TYPE>``.
* :prop_tgt:`INTERFACE_FILE_SETS_<TYPE>` to list ``INTERFACE`` and ``PUBLIC``
  file sets of type ``<TYPE>``.
